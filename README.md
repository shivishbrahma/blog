# Publog

Developer Blog of Shivishbrahma

## Themes

- Background: #161925
- Text: #f8f8f2
- Font: Libre Baskerville
- Language: English / Bengali

## Prequisites

- Ruby 3.3

## Installation

- Install all the dependencies using `bundle install`

- Start the server using `bundle exec jekyll serve`
